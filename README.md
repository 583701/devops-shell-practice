# DevOps Shell Practice Project


This is a basic shell script project created for DevOps practice.

The script (`system_info.sh`) displays:
- A welcome message
- The current date
- The system uptime

## How to Run

Make the script executable:
chmod +x src/system_info.sh
./src/system_info.sh
